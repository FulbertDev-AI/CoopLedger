# CoopLedger

Application décentralisée pour la gestion transparente des finances et votes des coopératives agricoles togolaises.

## Structure du projet
- `smart-contracts/` : Smart contracts Solidity (Ethereum Sepolia)
- `backend-laravel/` : API REST Laravel (MySQL)
- `frontend-web/` : Dashboard React.js (Vite)
- `mobile-app/` : Application mobile Flutter

Voir chaque sous-dossier pour les instructions spécifiques.
